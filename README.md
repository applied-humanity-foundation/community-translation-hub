## About

The Community Translation Hub is an open-source platform for collaboratively translating technology education materials into multiple languages. Language should never be a barrier to digital literacy. Starting with English and Chinese, we aim to make high-quality technology education accessible to learners worldwide.

## Features

- **Side-by-Side View** — Read original and translation together with paragraph alignment
- **Paragraph Linking** — Hover to highlight corresponding paragraphs across languages
- **Progress Tracking** — Visual indicators for translation completion status
- **Community Driven** — Anyone can contribute translations via pull requests
- **Quality First** — All translations reviewed by native speakers before merging
- **Offline Ready** — Download and use without internet

## Getting Started

```bash
# Clone the repository
git clone https://github.com/applied-humanity-foundation/community-translation-hub.git

# Open in your browser
open index.html
```

No build tools or dependencies required.

## Contributing

We welcome translators of all experience levels! Here's how to contribute:

### Adding a New Translation

1. Fork this repository
2. Choose a document that needs translation
3. Add your translation following the existing format
4. Submit a pull request

### Adding a New Language

1. Open an issue proposing the new language
2. Translate at least one complete document
3. Submit a pull request with your translation

### Translation Guidelines

- **Accuracy** — Translate meaning, not just words
- **Accessibility** — Use simple, clear language appropriate for beginners
- **Consistency** — Follow the glossary for technical terms
- **Cultural Sensitivity** — Adapt examples to be culturally relevant when appropriate
