    How to Contribute

    Join our community of translators and help make technology education accessible worldwide.

    1

    Fork the Repository

    Create your own copy of the project on GitHub so you can make changes freely.

    2

    Choose a Document

    Pick an untranslated document or improve an existing translation that needs revision.

    3

    Add Your Translation

    Translate the content using clear, accessible language that beginners can understand.

    4

    Submit a Pull Request

    Send your work back for review. A native speaker will review and help refine it.

    About the Foundation

    The Community Translation Hub is a project of the Applied Humanity Foundation, a 501(c)(3) nonprofit organization based in Colorado. Our mission is to advance human well-being through responsible AI application, open-source tools, and digital literacy education.

    We believe that language should never be a barrier to learning about technology. In a world increasingly shaped by digital tools and artificial intelligence, everyone deserves access to clear, accurate, and culturally sensitive educational materials in their own language.

    This platform is built and maintained entirely by volunteers. All content is freely available under open licenses for use by individuals, educators, schools, libraries, and community organizations worldwide.
